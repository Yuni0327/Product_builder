모델 정보

본 테스트는 Teachable Machine 기반 이미지 분류 모델을 사용합니다. 결과는 참고용이며 의료/진단 목적이 아닙니다.

  • 모델: Google Teachable Machine
  • 입력: 웹캠/이미지 업로드
  • 출력: 동물상 분류 확률

정책 및 고지

개인정보처리방침

업로드된 이미지는 브라우저에서만 처리되며 서버에 저장되지 않습니다.

이용약관

본 서비스는 참고용 결과를 제공하며, 결과에 대한 법적/의학적 책임을 지지 않습니다.

콘텐츠 정책
